== Description ==

**Flexi Product Slider & Grid for WooCommerce** is a lightweight, flexible, and fully responsive plugin that allows you to beautifully showcase your WooCommerce products using carousel, slider, and grid layouts.

Whether you're promoting featured products or displaying a curated collection, this plugin helps you enhance the shopping experience with minimal setup.

**Key Shortcode:**

`[flexipsg_carousel title="Product Carousel Title" category="" limit=""]`

**Shortcode Parameters:**

- `title` – Custom title for the carousel section.
- `category` – Product category slug to filter products (optional).
- `limit` – Maximum number of products to display (optional).

== Features ==

* Display products in a responsive carousel or grid.
* Filter products by category.
* Set a custom title and product limit.
* Fully mobile-friendly and compatible with all major themes.
* Easy-to-use shortcode – just copy and paste.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install via the WordPress plugin repository.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode `[flexipsg_carousel]` in your posts, pages, or widgets.

== Frequently Asked Questions ==

= Can I display products from a specific category? =  
Yes! Use the `category` attribute in the shortcode like this:  
`[flexipsg_carousel title="Sale Products" category="sale"]`

= What happens if I don’t set the `limit` attribute? =  
The plugin will show all available products.

= Is this plugin compatible with all themes? =  
Yes, it is designed to work with most WordPress themes. You may add custom CSS if needed.

= Can I display the products in grid instead of a slider? =  
The plugin supports both layouts. You can toggle layout display by plugin settings or shortcode (if supported in future versions).
